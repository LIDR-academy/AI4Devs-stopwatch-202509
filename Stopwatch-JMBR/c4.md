# C4 - stopwatch-JMB

## Level 1: Context

User opens a local/static web stopwatch in the browser. No server, login, database or external API is required.

## Level 2: Container

Single browser app:

- `index.html`: structure and accessible controls.
- `styles.css`: visual design based on the reference image.
- `script.js`: stopwatch/countdown logic and DOM controller.

## Level 3: Components

- `StopwatchEngine`: elapsed-time state.
- `CountdownEngine`: countdown-time state.
- `TimeFormatter`: `HH:MM:SS` + `mmm` formatting.
- `TimerApp`: UI orchestration and event handling.

## Why this architecture

The app is intentionally small. Separating pure timing logic from DOM logic makes it easier to test, maintain and review.
